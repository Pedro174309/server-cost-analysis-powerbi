# Server Cost & Consumption Analysis

Projeto de Business Intelligence focado na análise de custos e consumo de servidores.

## Objetivo
Identificar padrões de consumo e oportunidades de otimização de custos em infraestrutura de TI.

## Análises Realizadas
- Custo por servidor
- Consumo por período
- Identificação de picos de utilização
- Análise comparativa entre ambientes

## Tecnologias
- Power BI
- Modelagem de dados
- DAX

## Principais Insights
- Identificação de servidores com baixo aproveitamento
- Padrões de sazonalidade no consumo
- Oportunidades de redução de custos

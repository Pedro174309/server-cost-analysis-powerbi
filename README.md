# Server Cost & Consumption Analysis

Projeto voltado para análise de eficiência operacional e otimização de custos em ambiente de infraestrutura.

## Objetivo
Monitorar custos, analisar padrões de consumo de CPU e memória e identificar recursos subutilizados que possam gerar oportunidades de redução de despesas operacionais.

## Análises Realizadas
- Custo total e crescimento mensal
- Média de consumo de CPU e memória
- Evolução temporal dos custos
- Comparação de consumo por cliente
- Ranking de custo por cliente
- Identificação de picos de utilização e servidores subutilizados
- Análise comparativa entre ambientes

## Tecnologias
- Power BI
- Modelagem de dados
- DAX

## Principais Insights
- Identificação de clientes/servidores com maior impacto no custo total
- Relação entre consumo de recursos e custo operacional
- Padrões de variação mensal nos gastos
- Oportunidades de otimização via identificação de subutilização

## Preview do Dashboard
O dashboard foi estruturado com foco executivo, incluindo:
- KPIs estratégicos
- Análise temporal
- Comparativos de utilização
- Filtros interativos por cliente, servidor e período
